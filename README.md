# 🎓 Web Development Workshop Registration Form

A responsive and user-friendly registration form built using **HTML and CSS**. This project is designed to collect user details for a workshop and display a clean modern UI across all devices.

---

## 🚀 Features

- 📱 Fully Responsive Design (Mobile, Tablet, Desktop)
- 🎨 Clean and Modern UI
- 🔐 Form Input Fields with Icons
- 🧾 User-friendly Layout
- 🔄 Reset (Cancel) Button Functionality
- ⚡ Fast and Lightweight (No JavaScript)

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox & Media Queries)
- Font Awesome Icons
